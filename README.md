# My Ubuntu configuration

Copy-and-paste in terminal to clone this repository.
```
git clone https://github.com/Krittipoom/ubuntu-setup.git
```

Config (optional)
```
nano firstsetup.sh
```

Run First time setup
```
bash firstsetup.sh
```
